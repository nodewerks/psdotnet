# Contributing to PsDotNet

This repo is a one-way mirror of a private source-of-truth repository, used to host:

- The customer-facing landing page (this README)
- Sample / starter source code in `samples/`
- Built artifacts (Listener zip downloads) on the [Releases](https://github.com/nodewerks/psdotnet/releases) page
- A pointer to the [NuGet package](https://www.nuget.org/packages/PsDotNet) for the library itself

## What you can do here

- **Open issues** for bug reports, feature requests, or questions: https://github.com/nodewerks/psdotnet/issues
- **Read and copy from the samples** — they're MIT-licensed, use them however you like

## What we can't accept

- **Pull requests against this repo** — every push here comes from CI in the private repo, so any PR would be overwritten. Please open an issue describing what you'd change, and the maintainer will pick it up in the source repo.

## Reporting issues

When opening an issue, please include:

- The version of PsDotNet (NuGet) or PsDotNet.Listener (zip) you're using
- The version of Photoshop you're running
- A minimal repro if possible
- The full exception or error output

Thanks!
